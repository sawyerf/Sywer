# Sywer
HTTP server

## Installation
Linux:
- To install Sywer: `sudo sh setup.sh`
- If you want to run the server at the boot: `sudo systemctl start sywer`

## Settings
All the settings must be in the file `settings.swy`.
The differents settings are:
```
port	[PORT]
path	[SRC_PATH]
index	[FILE]
logs	[FILE]
error_[error]	[FILE]
```
